# Ethics and Scope

This repository is intended strictly for ethical cybersecurity education and authorized security testing.

## Authorized Environments

All testing documented in this repository is performed against:

- Systems I personally own
- Authorized virtual lab environments
- Intentionally vulnerable machines
- Capture-the-flag platforms
- Training platforms that permit security testing

## Rules I Follow

- I will not test systems without explicit permission.
- I will not expose private or sensitive information.
- I will sanitize screenshots, logs, IP addresses, and usernames.
- I will not upload employer or customer data.
- I will not publish credentials, API keys, or access tokens.
- I will not upload proprietary Hacker Academy course materials.
- I will document lessons in my own words.

## Repository Purpose

The purpose of this repository is to document my learning, hands-on practice, troubleshooting, and understanding of both offensive security techniques and defensive detection methods.
