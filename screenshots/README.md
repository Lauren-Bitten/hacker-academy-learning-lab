# Screenshots

This folder contains sanitized screenshots from authorized labs and personal test environments.

## Screenshot Rules

Before uploading a screenshot, verify that it does not contain:

- Employer names
- Customer information
- Real employee usernames
- Email addresses
- Credentials
- API keys
- Access tokens
- Internal hostnames
- Public IP addresses
- Sensitive file paths
- Personal information
- Proprietary course content

## What Screenshots May Include

- Authorized virtual lab environments
- Intentionally vulnerable machines
- Training platforms
- Commands executed in my personal lab
- Tool output from authorized systems
- Sanitized logs
- Sanitized network diagrams
- Lab results
